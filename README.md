# 🌅 Счётчик дней жизни — Telegram Bot

Бот отправляет каждое утро в **8:00 по Бишкеку** сообщение о том, сколько дней ты прожил.

## Команды бота
- `/start` — начать и ввести дату рождения
- `/today` — узнать счёт прямо сейчас
- `/setbirthday` — изменить дату рождения
- `/stop` — отключить уведомления

## Запуск на Railway.app (бесплатно)

### 1. Создай аккаунт
Зайди на https://railway.app и зарегистрируйся через GitHub

### 2. Создай новый проект
- Нажми **"New Project"**
- Выбери **"Deploy from GitHub repo"**
- Загрузи файлы бота (или используй GitHub)

### 3. Альтернатива — загрузить через GitHub
1. Создай репозиторий на https://github.com
2. Загрузи туда файлы: `bot.py`, `requirements.txt`, `Procfile`
3. В Railway подключи этот репозиторий

### 4. Готово!
Railway автоматически установит зависимости и запустит бота.

## Запуск локально (на своём компьютере)

```bash
pip install -r requirements.txt
python bot.py
```
